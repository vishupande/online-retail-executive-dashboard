# Online Retail Executive Dashboard

An interactive business intelligence dashboard created using retail sales data to analyze revenue trends, customer performance, country-wise sales, and global demand patterns.

## Features

* Monthly Revenue Analysis
* Top Revenue-Generating Countries
* Top Customers by Revenue
* Global Product Demand Analysis
* KPI Monitoring
* Interactive Visualizations

## Technologies Used

* HTML
* CSS
* JavaScript
* Chart.js

## Key Insights

* Strong seasonal growth observed in Q4.
* Netherlands emerged as the top international market.
* Top customers contributed a significant share of total revenue.
* Europe showed the highest product demand outside the UK.
